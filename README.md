# FS

Утилита для листинга содержимого каталога: замена `ls` с цветовой подсветкой по типам и отображением размеров файлов. Идея взята из [bat](https://github.com/sharkdp/bat) — как он дополняет `cat`, так FS дополняет `ls`.

Реализовано на C++17 (`std::filesystem`). Сборка — CMake.
