# Замена стандартной команды LS

FS - Утилита для отображения содержимого каталогов с цветовой подсветкой по типам и показом размеров файлов - современная замена `ls`
Идея вдохновлена **[bat](https://github.com/sharkdp/bat)**: как он дополняет `cat`, так FS дополняет `ls`

## Развитие проекта

Наработки и архитектурные решения, полученные при разработке FS (работа с файловой системой, форматирование вывода, обработка вложенных директорий и оптимизация обхода), впоследствии были использованы в проекте [Finder](https://github.com/FoKey-Junior/Finder), где получили дальнейшее развитие
